Honest constraints

Vision, voice, documents, web search, charts, code, translation, teaching run live on the Claude API in this single file.

Image generation — the Anthropic API understands images but does not render them. OmniLens crafts a production-ready generation prompt you can take to an image model. It does not draw the picture itself.

Video — handled as frame sampling: a few stills are pulled from your clip and analysed as images. There is no native audio-track analysis.

Speech-to-text relies on the browser's recognition engine. On iOS Safari this is limited; text input is always available as a fallback.
